Summary: Latitude Vs. X

This is a python script to visualize the weather of 500+ cities across the world of varying distance from the equator. after assembling the dataset, matplotlib was used to plot various aspects of the weather vs latitude. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations observed. Following relationships were studied:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
